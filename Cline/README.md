# Tools for Cline

This directory contains tools that can be used to support the development of Cline, an AI agent for software engineering tasks. 

## How to use the tools
1. Copy the file `Cline/.clinerules` to your project root directory. This file enables using of essential skills from `.github/skills` with Cline.
2. Copy content of 'Cline/.vscode/settings.json` to your project `.vscode/settings.json` file. This will constrain the Cline agent by its acces to sencible files in your project.
